## 快速开始

> 将下方代码粘贴至 head 标签

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/sedom/dist/sedom.min.css" />
```

> 将下方代码粘贴至 body 标签下

```html
<script src="https://cdn.jsdelivr.net/npm/sedom/dist/sedom.min.js"></script>
```
