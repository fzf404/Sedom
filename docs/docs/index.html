Sedom

快速开始

使用 CDN 加载
<link rel="stylesheet" href="https://cdn.fzf404.art/sedom/dist/sedom.css"/>
<script src="https://cdn.fzf404.art/sedom/dist/sedom.js"></script>
使用 NPM 安装
npm i sedom

在项目中使用

import Sedom from 'sedom'
import 'sedom/dist/sedom.css'

      
Sedom.initDOM()
